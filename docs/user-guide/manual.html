1第一次使用

這個系統可以幫你管理「進貨、賣東西、看庫存、看每月報表」這些事。打開瀏覽器（Chrome、Edge 都可以）就能用，不用裝任何東西。

怎麼打開系統

  1. 打開瀏覽器
    建議用 Chrome 或 Microsoft Edge。
  2. 輸入網址
    在網址列輸入 http://localhost:5173 然後按 Enter。
  3. 輸入帳號密碼
    你會看到一個登入畫面。輸入老闆／管理員給你的帳號和密碼，按 登入。
  4. 進到首頁
    登入成功就會看到一個有「本月銷貨額」「本月進貨額」「低庫存品項」等卡片的畫面。
小提示 忘記密碼？找系統管理員（admin）幫你重設，他在「使用者管理」可以幫你改新密碼。

3角色和你能做什麼

每個人有一個角色，角色決定你能看什麼、做什麼。共四種：

  • admin 系統管理員 全部都能做，只有他能新增 / 刪除帳號。
  • manager 主管 可以管商品、開進貨單、開銷貨單、確認入庫、確認出貨。
  • sales 業務 可以開銷貨單、確認出貨；商品和進貨只能看。
  • warehouse 倉管 可以「確認入庫」進貨單；其他大部分只能看。
注意 看不到的按鈕表示你的角色不能做那件事，這不是 bug。如果你覺得應該要能做，找你的主管或系統管理員調整。

進貨單跟銷貨單都是「兩階段」：

  • 先建立一張草稿（這時候還沒影響庫存）
  • 確認後變成已入庫或已出貨，這時候庫存才會動。
  • 不要的單可以變已取消，但只有草稿能取消。
記住 只有草稿才能改。一旦確認入庫或出貨就鎖住了 — 因為這時候已經影響庫存，亂改會錯亂。
5商品分類 adminmanager

把商品分成大類別，例如「飲料」「零食」「日用品」。先建分類，再建商品時選分類，比較好整理。

新增一個分類

  1. 左側選單點「商品分類」
  2. 右上角點 + 新增
  3. 輸入分類名稱
    例如「飲料」。
  4. 填說明（可不填）
  5. 按 儲存

編輯或刪除

在分類清單裡，每一列右邊都有「鉛筆」（編輯）跟「垃圾桶」（刪除）。

注意 刪掉分類後，原本屬於這個分類的商品「分類欄」會變空白，但商品本身還在。
6商品 adminmanager

這裡管你賣的東西。每個商品要有一個「商品編號（SKU）」和名稱。庫存量在這裡不能直接改 — 它會由進貨、銷貨自動算。

新增商品

  1. 左側選單點「商品管理」
  2. 右上角點 + 新增
  3. 填基本資料
    商品編號（例：A001）、名稱（例：可口可樂 330ml）、單位（個／瓶／包）。
  4. 選分類
    如果之前沒建分類，可以先去第 5 章建。
  5. 填售價跟成本
    售價是賣給客戶的價錢。成本不用填得很精準 — 之後每次進貨確認入庫時系統會自動更新。
  6. 低庫存閾值（選填）
    例如填 10，當庫存 ≤ 10 時系統會在庫存查詢顯示「低庫存」紅字提醒。填 0 表示不要提醒。
  7. 按 儲存
不要這樣做 不要為了調整庫存而修改商品。要調整庫存就建一張「進貨單」或「銷貨單」，這樣才有紀錄可查。

找商品

在最上面的搜尋框輸入商品編號、名稱、或條碼，會即時篩選。也可以用「分類」「狀態」下拉縮小範圍。

7供應商 adminmanager

記錄你從誰那邊進貨。建立進貨單之前要先有供應商。

新增供應商

  1. 左側選單點「供應商」
  2. 右上角點 + 新增
  3. 填供應商名稱
    這欄是必填，名稱不能跟既有的重複。
  4. 聯絡人、電話、Email、地址、統編、備註
    能填就填，方便日後對帳。
  5. 按 儲存
小提示 如果某個供應商已經不再合作，不要刪掉（會找不到歷史進貨單），把「狀態」改成停用就好。
8進貨單 adminmanagerwarehouse

進貨單是「向供應商買東西」的紀錄。流程：開單 → 等貨到 → 確認入庫（這時候庫存才增加）。

Step 1 — 開一張進貨單（草稿）

角色：adminmanager

  1. 左側選單點「進貨管理」
  2. 右上角點 + 新增
  3. 選供應商
  4. 加進貨項目
    每一列填：商品、數量、單價。可以點「新增一項」加更多商品。
  5. 備註（可不填）
  6. 按 儲存
    系統會給這張單一個編號，像 PO-20260514-0001。狀態是草稿。

Step 2 — 貨到了，確認入庫

角色：adminmanagerwarehouse

  1. 在進貨單清單找到那張草稿
  2. 點綠色勾勾「確認入庫」
  3. 確認對話框出現後按 確認

狀態變成已入庫，這時候：

  • 每樣商品的「庫存量」會自動加上這張單的數量。
  • 每樣商品的「成本」會被更新為這張單的單價（系統會記住最新成本）。

不要的單怎麼處理

只有草稿狀態才能取消。點紅色叉叉「取消單據」即可。

不能改 已入庫的進貨單不能再修改也不能取消。如果有錯，請建一張新的反向進貨單（負數量？不行）或新銷貨單把錯誤的量出掉 — 詳細請問主管。
9客戶 adminmanagersales

記錄你賣給誰。建立銷貨單之前要先有客戶。

新增客戶

  1. 左側選單點「客戶」
  2. 右上角點 + 新增
  3. 填客戶名稱
    必填，名稱不能跟既有的重複。
  4. 聯絡人、電話、Email、地址、統編、備註（看需要填）
  5. 按 儲存
10銷貨單 adminmanagersales

銷貨單是「賣東西給客戶」的紀錄。流程跟進貨單一樣分兩階段：開單 → 確認出貨（這時候庫存才減少）。

Step 1 — 開一張銷貨單（草稿）

  1. 左側選單點「銷貨管理」
  2. 右上角點 + 新增
  3. 選客戶
  4. 選業務員
    「業務員」是業績歸屬，跟「實際在輸入這張單的人」可能不同（例如主管幫業務 A 鍵單，業務員選 A）。如果你自己就是業務員角色，系統會預設選你自己。
  5. 加銷貨項目
    選商品時，單價會自動帶入商品本身的售價，可以再改。如果數量超過目前庫存，數量欄會變紅提醒你 — 但這時還沒扣庫存，存草稿不會被擋。
  6. 按 儲存
    系統會給這張單一個編號，像 SO-20260514-0001。狀態是草稿。

Step 2 — 出貨，確認

  1. 在銷貨單清單找到那張草稿
  2. 點綠色勾勾「確認出貨」
  3. 確認對話框出現後按 確認

狀態變成已出貨。系統會：

  • 檢查每個商品庫存是否夠 — 任何一個不夠都會整張單都不能出，並顯示哪些不夠多少。
  • 如果都夠，把每個商品的庫存量扣掉。
庫存不夠怎麼辦 錯誤訊息會告訴你哪個商品差多少。先去開一張進貨單把貨補進來、確認入庫，再回來確認這張銷貨單。
業務員 vs 建單者 系統會分別記下「業績算誰的」（業務員）和「實際是誰按鍵儲存的」（建單者）。所以主管代鍵單給業務，業績還是算給業務。
11查庫存 adminmanagersaleswarehouse

隨時查現在每個商品還剩多少。

  1. 左側選單點「庫存查詢」
  2. 看清單
    每一列：商品編號、名稱、分類、目前庫存、低庫存閾值、是否警示。
  3. 勾「只看低庫存」
    只看那些已經低於閾值的商品，方便決定要進什麼貨。
  4. 搜尋框
    可以按商品編號或名稱即時過濾。
小提示 低庫存的列會用淡黃色背景跟「低庫存」紅色 chip 標出來，一眼就看得到。
12月報表 adminmanagersaleswarehouse

看某個月每個商品的進銷情況。也可以匯出 Excel 拿給老闆看。

  1. 左側選單點「報表」
  2. 選年份、月份
  3. 看內容
    每個商品一列：期初庫存、本月進貨、本月銷貨、期末庫存、進貨金額、銷貨金額。最上方有「進貨合計」「銷貨合計」總數。
  4. 匯出 Excel（可選）
    點 匯出 Excel，瀏覽器會下載一個 .xlsx 檔，雙擊就能用 Excel 打開。
怎麼算的 只有「已入庫」的進貨和「已出貨」的銷貨會被算進去。草稿和已取消不會算。
13使用者帳號（系統管理員專用） admin

這一章只有 admin 看得到「使用者管理」。其他角色看不到也不能用。

幫同事開新帳號

  1. 左側選單點「使用者管理」
  2. 右上角點 + 新增
  3. 填基本資料
    帳號、姓名、Email、密碼（至少 8 字）。
  4. 選角色
    adminmanagersaleswarehouse 擇一。
  5. 按 儲存

幫人家改密碼

  1. 找到那個人，點鉛筆「編輯」
  2. 新密碼欄輸入新的密碼
    留空表示不改密碼。輸入了就會被替換。
  3. 按 儲存

停用一個人

編輯使用者，把「狀態」改成「停用」儲存。停用後他登不進來，但歷史單據還在。建議用「停用」而不是「刪除」。

注意 你不能刪除自己（避免不小心鎖住系統）。
14遇到問題怎麼辦

我打錯字了，怎麼救？

  • 還是草稿狀態：點鉛筆編輯就能改。
  • 已經已入庫／已出貨：不能改了。找主管處理。

確認出貨時被擋住，說庫存不夠

系統會告訴你哪幾個商品差多少。先去進貨單那邊建單、確認入庫補貨，再回來確認這張銷貨單。

「業務員」跟「建單者」有什麼差別？

  • 業務員 = 業績算誰的（用來算每月每人賣多少）。
  • 建單者 = 實際按下「儲存」這個動作的人。系統會自動記錄為當下登入的你，不用選。
  • 例：主管幫業務 A 鍵了一張單 — 業務員選 A，建單者自動是主管。月底算業績算給 A。

我看不到某個按鈕

你的角色沒有那個權限。例如sales 角色不能修改商品。如果這影響你做事，找主管把你的角色調整成需要的等級。

怎麼切換語言

右上角的「中文 ▾」可以切到 English。下次再開系統會記住你的選擇。

我登不進去

  • 確認帳號密碼有沒有打錯（大小寫要對）。
  • 確認帳號沒被「停用」。
  • 找系統管理員確認 / 重設密碼。

畫面看起來怪怪的

按 Ctrl + F5 重新整理（同時清快取）。還是怪的話，登出再登入。
